# 🏠 Global Airbnb Performance Dashboard – Power BI Project
DOWNLOAD DATASET FROM HERE - https://mavenanalytics.io/data-playground/airbnb-listings-reviews
## Overview
This project focuses on analyzing global Airbnb performance data using Power BI to uncover business insights related to listings, hosts, city-wise market share, customer reviews, ratings, seasonality, and trust indicators. The dashboard provides an interactive and visual representation of Airbnb trends across multiple cities and property types to support data-driven decision-making.

The analysis highlights important patterns such as listing growth trends, customer review behavior, city performance comparisons, and host credibility metrics.

## Dashboard

### 📊 Page 1: New Listings Performance

**Insights:**

- Airbnb reached the highest number of new listings around 2015.
- Growth slowed because of local regulations.
- COVID-19 caused a major decline in listing growth.

**Visuals Used:**

- KPI Cards
- Trend Line Charts
- Property Type Comparison

---

### 📊 Page 2: Market Share & Ratings Analysis

**Insights:**

- Paris, New York, and Sydney contribute significantly to total listings.
- Mexico City and Rio de Janeiro achieved higher overall ratings.
- Hong Kong and Istanbul received comparatively lower ratings.

**Visuals Used:**

- Market Share Analysis
- Pareto Chart
- Rating Comparison Chart
- Average Price Analysis

---

### 📊 Page 3: Reviews, Seasonality & Trust Analysis

**Insights:**

- Most users wrote only one review.
- Review activity changes seasonally across cities.
- More than two-thirds of Airbnb hosts are identity verified.

**Visuals Used:**

- Review Frequency Distribution
- Seasonal Trend Analysis
- Trust Indicator Dashboard
